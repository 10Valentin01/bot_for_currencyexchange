1. Папка data. В ней находятся запросы к базе данных.

2. Папка handlers. Имеет 2 файла users и admin.
Файл admin содержит в себе функции админ меню.
Файл users содержит в себе обработчики кнопок в меню

3. Папка settings. Имеет 5 файлов config, func, keyboards, state, text
Файл Config содержит в себе токен бота, channel_id канала и список user_id админов
Файл func содержит в себе функцию проверки подписки на канал
Файл keyboards содержит в себе кнопки, которые отображаются в пользовательском и админском меню
Файл state содержит в себе (классы)
Класс — в объектно-ориентированном программировании, представляет собой шаблон для создания объектов, обеспечивающий начальные значения состояний: инициализация полей-переменных и реализация поведения функций или методов
Файл text содержит в себе текста, которые отображаются в пользовательском меню

4. Файлы main и loader содержат в себе настройки для запуска бота. Их не трогать

5. Файл china.jpg это фотография, которая отправляется в стартовом сообщении

6. Файл main.db это файл базы данных, с которой взаимодействует бота

Для запуска бота нужно уставновить следующие пакеты: aiogram, sqlite3
